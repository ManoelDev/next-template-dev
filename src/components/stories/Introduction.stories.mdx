# Welcome to Storybook

O Storybook ajuda você a criar componentes de interface do usuário isoladamente da lógica de negócios, dos dados e do contexto de seu aplicativo.
Isso facilita o desenvolvimento de estados difíceis de alcançar. Salve esses estados de IU como **stories** para revisitar durante o desenvolvimento, teste ou controle de qualidade.
